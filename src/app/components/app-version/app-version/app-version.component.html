App de toma de datos

App para tomar datos sobre el terreno, almacenarlos en el dispositivo, y enviarlos al servidor cuando se tenga conexion a internet

Una vez suba los datos a internet, se le enviará un email para poder consultarlos.

Igualmente se se enviará un email a los técnicos para que puedan revisar los datos del predio, e iniciar el expediente.

Para subir datos al servidor necesita una cuenta, que puede obtener en este enlace

A todos los propietarios de los predios se les creará igualmente una cuenta para que puedan revisar sus datos y propiedades. Se les enviará un email de aviso, con un enlace para que establezcan su contraseña de acceso.

La versión actual permite lo siguiente:

  • Almacenar en el dispositivo los datos de los propietarios
  • Medir con GPS los perímetros del predio, y almacenarlos en el dispositivo
  • Almacenar varios predios en el dispositivo
  • Consultar los datos del dispositivo
  • Iniciar sesión en el dispositivo
  • Enviar los datos al servidor

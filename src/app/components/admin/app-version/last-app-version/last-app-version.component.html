App de toma de datos

App para tomar datos sobre el terreno, almacenarlos en el dispositivo, y enviarlos al servidor cuando se tenga conexion a internet

Una vez suba los datos a internet, se le enviará un email para poder consultarlos.

Igualmente se se enviará un email a los técnicos para que puedan revisar los datos del predio, e iniciar el expediente.

Para subir datos al servidor necesita una cuenta, que puede obtener en este enlace
